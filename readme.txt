This plugin helps remove the URL field parameter from the GeneratePress comment box with ease.

== Description ==
"URL Field Remover for GeneratePress" is a free plugin for GeneratePress theme, that helps to remove the URL or Website field from the comment form or comment box in GeneratePress theme.

Features:

1. Remove URL Field in both free and premium Version of GeneratePress theme's Comment Box.

== Installation ==
#### In the WordPress dashboard
1. Visit 'Plugins > Add New'
1. Search 'URL Field Remover for GeneratePress'
1. Check the plugin is from 'Bimal Raj Paudel'
1. Install and Activate the plugin.
1. After activation you don't have to do anything. 

#### Manually
1. Download the file <a href="https://wordpress.org/plugins/url-field-remover-for-generatepress/" target="_blank">From here.</a>
1. Unzip the file.
1. Upload the folder URL Field Remover for GeneratePress into 'wp-content/plugins/' folder.
1. Activate.

== Frequently Asked Questions ==
= How does the plugin work? =
The plugin injects the code directly to functions.php to remove URL Field from the comment form in GeneratePress Theme.

= Will this plugin work for free version of GeneratePress theme? =
Yes, it works for both free and premium GeneratePress theme. This plugin even works with other themes as well but it is advised to use only for GeneratePress.

= Will this break my theme? =
It has been tested with various versions of GeneratePress theme and should not break your theme unless it collides with any other plugins.

= Do you need to do anything after installing the plugin? =
No, you don't have to do anything. Just install and activate the plugin and it will do everything for you.
